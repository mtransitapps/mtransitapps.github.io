Privacy Policy

MTransit Apps ("we", "us", "our”, "developers") has developed this version of the mobile application "MonTransit" ("this app").

The old version of the mobile application "MonTransit" was developed by "Mathieu Méa".

1. Consent

By downloading, installing and using this version of the mobile application "MonTransit", the user accepts the terms and conditions described of this Privacy Policy.

2. Scope

This privacy policy does not cover third parties apps, products, services or websites contained inside this mobile application (such as Google Mobile Ads).

3. Information collected

This app anonymously tracks and reports user's activity to third party services (such as Google Analytics):

  • device location (city-level precision)
  • user interactions (such as screens displayed, buttons clicks)
  • user preferences (such as distance unit)
  • bugs and crashes
  • device location: approximate & precise
  • device ID: user-resettable and user-deletable Advertising ID
  • app interactions (such as screens displayed, buttons clicks)
  • user preferences (such as distance unit)
  • app info and performance, crash logs and diagnostics

No user data is being collected by the MonTransit app when communicating with YouTube API Services.

4. Information usage

The information collected by this mobile application might be used by third party services (such as Google).

The information collected is used to improve this mobile application based on the usage of this mobile application by users.

5. Users' choice

Users can disable part of the collection of information by:

  • paying to remove advertising from this app
  • disabling the device app backup service

Users can disable entirely the collection of information by:

  • downloading the source code of this mobile application and disabling the code responsible for the collection of information
  • uninstalling this mobile application

6. Information sent to third parties

No user information is disclosed directly to third parties.

The app requests information from 3rd party data providers such as YouTube (Google).

By using the app, users are agreeing to be bound by the YouTube Terms of Service and Google Privacy Policy.

  • YouTube Terms of Service (ToS): https://www.youtube.com/t/terms
  • Google Privacy Policy: https://policies.google.com/privacy

The MonTransit app uses YouTube API Services to show you the latest videos from transit agencies.

7. Security

The collected information is stored in third parties services (such as Google Analytics).

Only Google Accounts with 2-Step Verification or strong passwords have access to this data.

8. Information retention

Collected information is retained in third party services for as long as possible.

9. Information access and removal

It's currently not possible to access or remove collected information since it's not possible to identify a specific user information.

10. Updates

This privacy policy might be updated as frequently as necessary.

11. Contact

For questions about this privacy policy, please send an email to: mtransit.apps+privacypolicy@gmail.com.
